## Overview

This project is an interactive Sales Dashboard created using Microsoft Excel.

## Features

- KPI Cards
- Monthly Sales Trends
- Daily Sales Trends
- Weekly Sales Trends
- State Wise Sales
- City Wise Sales
- Food Category Analysis
- Interactive Slicers

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
